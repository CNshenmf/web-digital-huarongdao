# web-digital-huarongdao
# 网页端数字华容道（卡通护眼版）
一款适配iPhone、华为手机、iPad（横竖屏）的网页端数字华容道游戏，采用分层设计（HTML/CSS/JS），支持GitHub版本控制，主打卡通可爱风+防蓝光护眼配色，无广告、无音效、无多余功能，聚焦核心游玩体验。

## 项目介绍
- 适配终端：iPhone、华为手机、iPad（支持横竖屏自由切换）
- 核心功能：4档难度（3×3~6×6）、通关提示、步数/耗时记录（当前会话）、游戏重置
- 设计风格：卡通可爱风，防蓝光护眼配色，极简纯净体验
- 技术栈：HTML5 + CSS3 + JavaScript（原生，无框架，轻量化）
- 分层设计：严格区分结构、样式、交互，便于维护和多人协作

## 启动方式
1. 克隆仓库：`git clone https://github.com/你的用户名/web-digital-huarongdao.git`
2. 进入项目根目录，用浏览器打开 `src/html/index.html` 即可启动游戏
3. 无需安装依赖，轻量化加载，支持离线缓存（打开一次后断网可玩）

## 功能说明
1. 难度选择：3×3（入门）、4×4（进阶）、5×5（高阶）、6×6（挑战）
2. 核心玩法：点击/滑动空白块相邻数字，按顺序排列完成解谜
3. 辅助功能：通关提示（仅1种提示方式）、重新开始、返回难度选择
4. 成绩记录：实时显示当前步数、耗时，记录各难度当前会话最短步数和耗时（不本地缓存）
5. 适配说明：iPad支持横竖屏自由切换，自动适配不同屏幕尺寸

## 编码规范
- 命名规范：文件（小写+连字符）、变量/函数（小驼峰），语义化命名
- 注释规范：关键逻辑、工具函数添加详细注释
- 提交规范：feat（新增功能）、fix（修复问题）、style（优化样式）、docs（更新文档）
